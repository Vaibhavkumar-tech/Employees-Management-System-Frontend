🚀 Employees Management System

A full-stack web application designed to manage employees, assign tasks, and track work progress efficiently.
It provides separate dashboards for Admin and Employees with a modern, responsive UI. 💼✨

🌟 Key Features

  ✅ Secure Login Authentication for Admin & Employees
 
  ✅ Admin Dashboard to assign and manage tasks

  ✅ Employee Dashboard to view assigned tasks
  
  ✅ Task Status Tracking (🆕 New, ⚡ Active, ✅ Completed, ❌ Failed)
  
  ✅ Employee Details Management
  
  ✅ Responsive and User-Friendly Interface 📱💻
  
  ✅ Real-time Data Storage with MongoDB 🍃

🛠️ Tech Stack

🎨 Frontend React.js ⚛️ Tailwind CSS 🌬️

⚙️ Backend Node.js 🟢 Express.js 🚂 🗄️ Database MongoDB 🍃
